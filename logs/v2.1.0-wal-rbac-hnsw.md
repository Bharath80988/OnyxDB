# v2.1.0 (Write-Ahead Log, RBAC, HNSW Vector Search)

## Features Added (feat)
- **WAL & Crash Recovery**: Introduced `WriteAheadLog` to guarantee database durability. The B+ tree automatically replays logs on system crash recovery.
- **Role-Based Access Control (RBAC)**: Enforced a security layer in `QueryController` rejecting inserts from `READ_ONLY` profiles (returning 403 Forbidden).
- **HNSW Vector Index**: Built a native Java flat Exact KNN vector search placeholder for future HNSW hierarchical graph expansions. Integrated spatial queries to compute cosine distances natively in-memory without dependencies.
- **Architecture Documentation**: Generated an end-to-end mapping document covering all folder components and responsibilities in `status/architecture.md`.

## Bug Fixes (bug)
- Resolved Docker `maven-bin` conflict and missing dependencies. Verified clean `mvn clean install` passes all tests with zero errors.
