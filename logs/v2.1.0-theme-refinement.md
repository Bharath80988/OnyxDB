# v2.1.0 - Theme Refinement & Optimization

## Overview
In version 2.1.0, we simplified the visual language of the OnyxDB dashboard, moving away from an excessive multi-theme approach in favor of a polished, refined Light and Dark mode that better suits the OnyxDB brand identity.

## Changes

### 1. Unified Earthy Theme
- Removed the 10 DaisyUI themes that caused UI bloat and inconsistent visual states.
- Replaced the vibrant purple (`#8b5cf6`) with an elegant, earthy **Emerald Green** (`#10b981`).
- Replaced the cyan secondary color with a warm **Amber/Light Brown** (`#d97706`).
- This produces a sleek, organic "Onyx" aesthetic. 

### 2. Streamlined Navigation
- Replaced the Theme Dropdown menu in the `<Navbar />` with a simple, functional Sun/Moon toggle button.
- Fixed severe contrast issues that caused text to disappear on hover within the dropdown menu by removing the clashing background states.

### 3. Scroll Enhancements
- Fine-tuned the `Lenis` smooth-scrolling engine in `App.tsx`.
- Increased the scroll speed (reduced duration from `1.2` to `0.6`), providing a much snappier and responsive experience while retaining cinematic smoothness.

### 4. Status Tracking Updates
- Updated the open-source Status Page roadmap to reflect the transition from "10 DaisyUI Themes" to a "Refined Earthy Theme".
