# v1.3.0 - Omni-Channel Architecture & Containerization

## 🚀 Features
- **Visual Node Builder:** Introduced a React Flow interface in the dashboard for visual query construction.
- **Terminal CLI:** Added a Python SDK (`sdks/python/onyxdb.py`) for command-line querying.
- **Redis Caching:** Integrated `@Cacheable` Spring Data Redis to dramatically improve `SELECT` query throughput.
- **Dockerization:** Unified the React frontend and Java backend into a single portable Docker image via a multi-stage `Dockerfile`.
- **GitHub Actions:** Implemented an automated CI/CD pipeline for testing and building.

## 🛠 Fixes & Improvements
- Updated `README.md` with JitPack installation instructions for embedding `onyxdb-core`.
- Refactored `QueryController` to route through a new caching `QueryService`.
