# OnyxDB Checkpoint v0.2.0 - Base Working Version

**What we have done:**
- Initialized core storage directory structures (`database/`, `wal/`).
- Implemented `Page` (8KB fixed-size data structure).
- Implemented `StorageManager` using Java NIO for high-performance reading/writing of pages.
- Implemented `BufferPool` with an LRU replacement policy for caching pages in memory.
- Implemented `WriteAheadLog` (WAL) for append-only sequential writes to guarantee durability.
- Wrapped the storage engine in the `onyxdb-api` Spring Boot application.
- Initialized `/query` endpoint placeholder.

**What is to be done:**
- Verify with unit tests.
- Integrate the API into the React dashboard.
